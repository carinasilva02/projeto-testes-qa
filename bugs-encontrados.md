# Registro de Bugs 
## BUG001 - Login permite senha inválida

Sistema testado:
Swang labs 

Severidade:
Alta

Prioridade:
Alta 

Descrição:
O sistema permite tentativas de login com uma senha inválida.

Passos para reproduzir:
1- Acessar a página login do Swag Labs
2- Inserir username de "standar_user."
3- Inserir senha "123456"
4- Clicar no botão Login 

Resultado esperado:
O sistema deve impedir o login e exibir mensagem de erro.

Resultado atual:
O sistema apresenta uma mensagem de erro informando que as credenciais.

Status: 
Bug resgistrado para análise
